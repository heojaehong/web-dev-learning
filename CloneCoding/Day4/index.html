CloudDocs 개요

이 문서는 CloudDocs 플랫폼의 전반적인 구조와 기능을 설명합니다. CloudDocs를 처음 사용하시는 분들을 위해 핵심 개념과 시작하는 방법을 안내합니다.

💡 참고

신규 사용자는 빠른 시작 가이드를 먼저 읽어보시는 것을 권장합니다.

주요 기능

CloudDocs는 다음과 같은 핵심 기능을 제공합니다:

  • 자동 배포 - Git 저장소와 연동하여 자동으로 배포합니다.
  • 확장성 - 트래픽에 따라 자동으로 스케일링됩니다.
  • 모니터링 - 실시간 메트릭과 로그를 확인할 수 있습니다.
  • 보안 - SSL 인증서가 자동으로 적용됩니다.

CloudDocs를 사용하기 위한 최소 요구사항입니다.

메모리 512MB 2GB 이상
